# LinkedIn Project Descriptions

## Threat Actor Analysis and Physical Security Risk Mitigation

Analyzed common cyber and physical threat actors, their motivations, organizational assets at risk, and practical controls to reduce unauthorized access. The project focused heavily on tailgating, social engineering, access control, employee awareness, and physical security safeguards.

**Skills:** Threat actor analysis, Risk assessment, Access control, Physical security, Security awareness, Incident response planning

## IT Service Desk Troubleshooting and Knowledge Base Development

Created a structured IT support knowledge base covering enterprise incidents such as SharePoint access failures, Windows blue screens, password resets, printer issues, Wi-Fi connectivity problems, software access requests, and authentication failures.

**Skills:** ITSM, Help desk ticketing, Windows 10 support, SharePoint, DocuSign, Adobe Acrobat Pro, Wi-Fi troubleshooting, Knowledge base documentation

## Olympic Basketball Python Text-Based Adventure Game

Developed a command-line Python adventure game where a player navigates an Olympic basketball tournament, collects required items, manages inventory, avoids a hazard room, and reaches the victory condition.

**Skills:** Python, Command line interface, Dictionaries, Lists, Loops, Functions, Conditional logic, Input validation

## Enterprise Network Infrastructure Design

Designed a secure and scalable network infrastructure for a new office location, including fiber connectivity, wireless access points, routers, switches, firewalls, VPN support, VLAN segmentation, IP addressing, QoS, and monitoring.

**Skills:** Network design, OSI model, VLANs, Private IP addressing, Subnetting, VPN, QoS, Routers

## Cisco Packet Tracer Multi-Site Network Design and Routing Implementation

Built, configured, and documented a multi-site network topology in Cisco Packet Tracer using routers, switches, servers, PCs, printers, IP addressing, serial links, static routing, and RIP routing.

**Skills:** Cisco Packet Tracer, IPv4, Subnetting, Static routing, RIP, Routers, Switches, WAN links

## Cyber Defense Strategy for IoT and Mobile-Connected Systems

Assessed cybersecurity risks for a manufacturing organization deploying smart headset technology connected through Bluetooth, mobile devices, and central servers. Developed a layered security strategy covering human risk, data protection, MDM, endpoint security, encryption, and incident response.

**Skills:** IoT security, Mobile security, MDM, Endpoint security, Encryption, Data classification, KMS, HSM

## Network Protection Strategy and IDS/IPS Security Architecture

Evaluated network protection technologies for a segmented enterprise environment and recommended secure design principles, IDS/IPS monitoring, and operational response procedures.

**Skills:** IDS, IPS, Snort, Suricata, Network segmentation, Least privilege, Fail-safe defaults, Security monitoring

## Healthcare Network Security and HIPAA Compliance Assessment

Assessed a healthcare network security plan with a focus on confidentiality and PHI protection. Identified wireless and administrative protocol vulnerabilities and developed HIPAA-aligned remediation strategies.

**Skills:** HIPAA, PHI, WPA3-Enterprise, VPN, VLANs, SSH, SFTP, MFA

## Financial Web Application Security Assessment

Assessed a financial services web application for insecure communications, XML weaknesses, client-side authentication risks, and web application vulnerabilities. Recommended secure design controls before deployment.

**Skills:** Application security, OWASP, TLS/HTTPS, XML security, XSS, Input validation, Output encoding, Session management

## Linux System Hardening Automation and PKI Implementation

Automated Linux security hardening tasks and implemented a Public Key Infrastructure using OpenSSL. The project included Bash scripting, security baseline development, certificate authority creation, CSR handling, and operating system configuration checklists.

**Skills:** Linux, Ubuntu, Bash, OpenSSL, PKI, Certificate Authority, CSR, Security baselines

## Enterprise Access Control and Security Assessment for Corporate Acquisition

Conducted a post-acquisition cybersecurity assessment for Grey Matter and BrainMeld, combining an executive security briefing with an access control compliance review. The project evaluated identity governance, remote access, Active Directory privileges, vendor access, file sharing, MFA, SSO, and account lifecycle risk.

**Skills:** IAM, Active Directory, MFA, VPN security, SSO, RBAC, Least privilege, Access reviews

## Enterprise Network Segmentation and Firewall Security Architecture

Restructured an enterprise network using segmentation, VLAN assignments, dedicated switches, firewall rules, and controlled traffic flows. The project allowed only required HTTP/HTTPS access to the web server while blocking unauthorized WAN access to other systems.

**Skills:** Network segmentation, VLANs, Firewall rules, Access control lists, WAN security, Switch configuration, Traffic filtering, CIA triad

## Digital Forensics Investigation and Evidence Preservation

Developed a digital discovery and evidence preservation strategy for a suspected Acceptable Use Policy violation and possible connection to a cybersecurity incident. The project focused on evidence sources, forensic imaging, chain of custody, Autopsy, FTK Imager, and write-blocker usage.

**Skills:** Digital forensics, FTK Imager, Autopsy, Write blockers, Chain of custody, Log analysis, Network forensics, Forensic imaging

## Risk-Based Cybersecurity Decision Making and Enterprise Risk Assessment

Developed a risk-informed cybersecurity decision-making framework using risk registers, Business Impact Analysis, NIST, CIS Controls, objective scoring, breach lessons learned, and security metrics.

**Skills:** Risk registers, BIA, NIST, CIS Controls, Security metrics, Risk scoring, MFA, Security governance

## Enterprise Cybersecurity Program Planning and Stakeholder Communication Management

Created a multi-phase cybersecurity project plan using a Gantt chart and communication plan. The initiative covered assessment, policy development, process improvement, technology deployment, infrastructure hardening, documentation conversion, and stakeholder communications.

**Skills:** Project management, Gantt charts, Stakeholder communication, Patch management, Authentication policy, Disaster recovery, Endpoint protection, Centralized monitoring

## Healthcare SLA Security Control Implementation and Compliance Strategy

Developed a security control plan for Fit-vantage Technologies to meet SLA requirements with Helios Health Insurance. The plan used CIS Controls Version 8 to address data protection, secure system configuration, incident response, phishing defense, and audit readiness.

**Skills:** CIS Controls v8, SLA compliance, Healthcare security, Data protection, Encryption, Secure configuration, Patch management, Incident response
